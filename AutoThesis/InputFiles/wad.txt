Web App Development:
Research into web application development showed a focused effort on building a Streamlit-based platform for AI-driven loan risk assessment and governance. The project integrated a TabNet deep learning model with specific business rules and language model explanations. User input was gathered through ten distinct numeric fields including Annual Income, Credit Score, Debt-to-Income Ratio, Bankruptcy History, Previous Loan Defaults, Length of Credit History, Monthly Income, Net Worth, Interest Rate, and Total Debt-to-Income Ratio. Upon triggering the prediction, the system processed six engineered features and scaled chosen data columns with a pre-saved StandardScaler from a designated file. The processed data was then passed to the TabNet model stored in a specific zip file to generate a risk score. Business rules applied a threshold of 50% to mark high-risk cases as Rejected despite initial model outcomes. An explanation feature used OpenAI GPT-4o-mini with a detailed prompt to create a formal report covering risk analysis, financial insights, and improvement suggestions. The interface in Streamlit presented the risk score, initial model verdict, business rule flag, final verdict, and the complete AI-generated report directly on the screen for user review.
