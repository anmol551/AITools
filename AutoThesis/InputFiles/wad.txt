Web App Development:
The web application for loan risk assessment employed Streamlit as the framework in its development. Users interacted with the interface through ten specific numeric input fields that captured key financial data such as Annual Income Credit Score Debt-to-Income Ratio Bankruptcy History Previous Loan Defaults Length of Credit History Monthly Income Net Worth Interest Rate and Total Debt-to-Income Ratio. Upon clicking the designated button the application processed inputs by incorporating six engineered features and applying a pre-fitted StandardScaler configuration loaded from a file to prepare the data. This scaled data fed into the saved TabNet model which generated a continuous risk score based on its regression capabilities. The system then evaluated the score against a threshold where values above 50 led to a High Risk classification and an automatic override to a Rejected status irrespective of the initial model output. Following this the application invoked OpenAI's GPT-4o-mini with a structured prompt to formulate a comprehensive underwriting report that included analysis of risk interpretation comparisons between model predictions and governance rules identification of financial strengths key risk drivers suggestions for improvements and an executive summary. The interface presented the predicted risk score the model's decision the business rule flag the final decision and the complete AI-generated report directly to users for immediate review. This implementation showcased effective integration of machine learning and rule-based systems in a practical setting.
