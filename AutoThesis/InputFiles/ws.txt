**Summary of AI Text Detection Web Application**

**Overview**: The web application, built with Dash, enables users to classify text as human-written or AI-generated using a pre-trained Random Forest model. It provides a user-friendly interface for text input, preprocessing, and prediction, displaying results with confidence scores and a visual confidence bar.

**Python Framework Used**: Dash (with `dash_bootstrap_components` for styling).

**Libraries & Dependencies**: `dash`, `dash_bootstrap_components`, `pandas`, `re`, `string`, `contractions`, `emoji`, `pickle`, `num2words`, `nltk` (`word_tokenize`, `stopwords`, `WordNetLemmatizer`), `sentence_transformers`.

**User Input Handling**: Users input text via a `dbc.Textarea` component, styled with a placeholder and a modern design, submitted through an "Analyse Text" button.

**Main Functions & Features**: The app features three preprocessing functions: `basic_clean_text` (lowercase, punctuation removal), `advance_clean_text` (expands contractions, removes emojis, converts numbers to words), and `stop_lemma_clean_text` (tokenization, stopword removal, lemmatization). The UI includes a gradient-styled title, description, input section, and result cards with confidence indicators.

**Prediction Function**: The `process_text` callback processes input via a pipeline: text is cleaned using the three functions, encoded into semantic embeddings with `SentenceTransformer` (`all-MiniLM-L6-v2`), and classified by a Random Forest model (`rf_model`). It outputs a label ("Human-Written" or "AI-Generated") with confidence scores (e.g., 80%+ as "High") and a styled confidence bar.
